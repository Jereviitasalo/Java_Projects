Toteutin Java kielellä kiinteistöjen vakuutustietoja sisältävän tietosäiliön.
Kiinteistöä kuvaa luokka Property, jolla on attribuutteina kiinteistön tyyppi ja
sijaintipaikka (kummatkin ovat String-tyyppisiä).
Vakuutustietoja kuvaa luokka InsuranceInfo, jonka yhtenä attribuuttina on Propertyluokan olio
ja toisena attribuuttina kiinteistön vakuutusarvo (double).
Tietosäiliöluokka InsInfoContainer tallentaa vakuutustieto-olioita ArrayListkokoelmaan.
